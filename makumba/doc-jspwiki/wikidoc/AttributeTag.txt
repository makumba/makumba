[{Table

||Name||Required||Request-time||Description||Comments 

|name 
|true 
|false 
|the name of the attribute. Will be printed if var is not present and there is no exception when finding the argument. If there is an exception but exceptionVar is indicated, nothing will be printed. 
|Fixed, Mandatory. 

|var 
|false 
|false 
|The name of the java.lang.Object variable where to store the attribute value. Note that you will have to cast it to its specific type. 
|Fixed, Mandatory. 

|exceptionVar 
|false 
|false 
|The name of the created java.lang.Throwable variable that will hold eventual exceptions raised when trying to compute the attribute. If exceptionVar is missing, any raised exception will stop page execution. If it is present, and AttibuteNotFoundException occurs, the attribute is set to null (so a next request for the attribute will return null) 
|Fixed 

|db 
|false 
|true 
|The database in which the business logic (if any) will compute the attribute. The default database is normally used. 
|Fixed 


}]
