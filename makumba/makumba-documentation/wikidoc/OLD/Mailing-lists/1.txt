makumba has several mailing-lists, hosted at SourceForge

!! makumba-user@lists.sourceforge.net

If you need help, have questions, comments, etc., [subscribe to the users mailing-list|https://lists.sourceforge.net/lists/listinfo/makumba-user]

!! makumba-devel@lists.sourceforge.net

If you'd like to get in touch with the developers, feel free to [contact them|mailto:makumba-developers@lists.sourceforge.net].

If you want to take part in makumba development, you can [request to be added to the developers mailing-list|https://lists.sourceforge.net/lists/listinfo/makumba-devel] (if you do so, please send us a mail beforehand so we know you're not a spam-bot :-) )

!! makumba-comm-build@lists.sourceforge.net

If you're interested in helping out with building the makumba community, you can [subscribe to the community building mailing-list|https://lists.sourceforge.net/lists/listinfo/makumba-comm-build]
