Struts extension for multiple forms

Multiple forms is one of the major strengths of makumba. However, the flagship Web MVC framework Struts does not consider multiple forms. This is one of the reasons why we cannot adopt Struts at the moment.

Brief:
  • look at Struts and think how/if it needs to be extended to be able to use multiple forms
  • if that's doable, consider how Makumba can be adapted to use Struts
  • look also at controller-model separation
